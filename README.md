# 📅 Guild Bot - Telegram бот для управления расписанием гильдии

Бот для автоматических напоминаний о событиях. Разрабатывалось для гильдии, но можно адаптировать и для других напоминаний. Отправляет утренние дайджесты(план на день), напоминания о событиях, уведомления о старте, а также предоставляет команды для просмотра расписания и создания опросов.

## ✨ Возможности

- 🌅 Ежедневный утренний дайджест с расписанием на день
- 🔔 Напоминание о событии за 11 минут до начала
- ⚔️ Уведомление о старте события
- 📝 Автоматическое удаление старых сообщений
- 📊 Просмотр расписания на сегодня/неделю/ближайшее событие
- 🗳️ Создание опросов для сбора команд (лес, арена - спец. функционал для игры)
- 📁 Хранение расписания в CSV (легко редактируется в Excel или любом текстовом редакторе)

## 🚀 Быстрый старт

### 1. Установка Python

**Windows:**
1. Перейдите на [python.org/downloads](https://www.python.org/downloads/)
2. Скачайте Python 3.10 или новее
3. **ВАЖНО:** при установке обязательно поставьте галочку "Add Python to PATH"
4. Проверьте установку: откройте командную строку (Win+R → cmd) и введите:
   ```bash
   python --version
   ```
   Должно показать версию Python

**macOS:**
```bash
brew install python3
```
или скачайте с официального сайта

**Linux (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install python3 python3-pip
```

---

### 2. Создание бота в Telegram

1. Найдите в Telegram [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Придумайте имя бота (например, `Guild Schedule Bot`)
4. Придумайте username (должен заканчиваться на `bot`, например `guild_schedule_bot`)
5. **Сохраните полученный токен!** Он выглядит примерно так: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`
   
6. Так же в BotFater можно настроить меню для бота (список команд представлен ниже)

---

### 3. Получение ID чата и темы

1. Добавьте бота в свою группу/чат
2. Необходимо назначить бота администратором
3. Отправьте команду `/getID` в чате с ботом
4. Бот ответит:
   ```
   🆔 ID чата: -1234567890909
   📌 ID темы: 5 (если используется тема)
   👤 Отправитель: 123456789
   ```
5. **Сохраните `ID чата` и `ID темы`** (если используете тему)

### 6. Настройка конфигурации

Откройте файл `config.py` и укажите свои данные:

```python
BOT_TOKEN = "ваш_токен_от_BotFather"           # Токен бота (из пункта 2)
CHAT_ID = -1234567890909                        # ID вашей группы или диалога
THREAD_ID = 5                                   # ID темы (если используете) или None (если группа без тем или это диалог)
```

Если картинка лежит в другом месте, поправьте пути:

```python
IMAGE_PATHS = {
    'boss_1': "C:/Path/to/image/boss_1.jpg",  # полный путь
    # рекомендуется хранить изображения в папке images и использовать относительный путь:
    'boss_1': os.path.join(IMAGES_DIR, "boss_1.jpg"),
}
```

### 7. Создание расписания

Создайте файл `schedule.csv` в корне проекта (или скопируйте пример):

```csv
name,day,time,description,image_key,is_notification
Ги боссы (Breaking Army),2,20:00,Сражение с гильдейскими боссами,boss_1,False
Ги боссы (Breaking Army),5,20:00,Сражение с гильдейскими боссами,boss_2,False
Ги арена (Test Your Skills),3,20:00,Бои на гильдейской арене,arena,False
Гильд пати,0,20:00,Сбор на базе гильдии,guild_party,False
Напоминание о регистрации на войну,1,10:00,Пост <a href='https://t.me/c/link_to_the_message/11/22'> как регистрироваться</a>,,True
```
(если папка не создана она создастся автоматически)

**Расшифровка полей:**
- `name` - название события
- `day` - день недели (0=понедельник, 1=вторник... 6=воскресенье)
- `time` - время в формате HH:MM
- `description` - описание (можно использовать HTML-теги, например `<a href="ссылка">текст</a>`)
- `image_key` - ключ картинки из `IMAGE_PATHS` (оставьте пустым если не нужно)
- `is_notification` - если `True`, отправляется только одно уведомление (без напоминания за 11 минут)

## 🛠️ Управление расписанием

### Как добавить новое событие

1. Откройте `schedule.csv` в Excel или любом текстовом редакторе
2. Добавьте новую строку:
   ```
   Новое событие,3,19:30,Описание события,boss_1,False
   ```
3. Сохраните файл
4. В Telegram отправьте команду `/reload` (если добавили) или перезапустите бота

### Как изменить время события

1. Откройте `schedule.csv`
2. Измените время в нужной строке
3. Перезапустите бота или используйте `/reload`

### Как добавить нотис (одноразовое напоминание)

Добавьте строку с `is_notification=True`:
```
Напоминание о сборе,2,15:00,Не забыть собраться в 15:00,,True
```

## 🔧 Возможные проблемы и решения

### Бот не запускается
- Проверьте, установлен ли Python: `python --version`
- Установите зависимости: `pip install pytelegrambotapi apscheduler`
- Проверьте токен в `config.py`

### Бот не отвечает в группе
- Проверьте правильность `CHAT_ID` и `THREAD_ID` (если используется) в config.py
- Убедитесь, что бот добавлен в группу в качестве АДМИНИСТРАТОРА

### Картинки не отправляются
- Проверьте, что пути к файлам в `IMAGES_DIR` правильные
- Убедитесь, что файлы существуют

### Сообщения не удаляются
- Бот должен быть администратором в группе
- У бота должно быть разрешение на удаление сообщений
